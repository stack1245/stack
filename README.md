# 디스코드 프로필 관리 봇

유저의 프로필 정보를 등록하고 조회할 수 있는 디스코드 봇입니다.

## 기능

### 일반 사용자
- 프로필 등록/수정 (닉네임, 출생년도, 성별, 지역)
- 프로필 조회

### 관리자 전용
- 경고 추가/제거
- 관리자 메모 작성
- 경고 횟수 및 메모 조회

## 설치

```bash
# 패키지 설치
pip install -r requirements.txt

# 환경 변수 설정
cp .env.example .env
```

`.env` 파일에 봇 토큰 입력:
```
DISCORD_BOT_TOKEN=your_bot_token_here
```

## 실행

```bash
python main.py
```

## 명령어

### `/프로필등록`
프로필 등록 또는 수정

```
/프로필등록 닉네임:아키 출생년도:08 성별:남 지역:서울
```

### `/정보`
프로필 조회

```
/정보
/정보 유저:@username
```

### `/관리 경고추가`
경고 추가 (관리자 전용)

```
/관리 경고추가 유저:@username 횟수:1
```

### `/관리 경고제거`
경고 제거 (관리자 전용)

```
/관리 경고제거 유저:@username 횟수:1
```

### `/관리 메모작성`
관리자 메모 작성 (관리자 전용)

```
/관리 메모작성 유저:@username 메모:주의 필요
```

## 데이터 저장

모든 데이터는 SQLite 데이터베이스(`user_profiles.db`)에 저장되며, 봇 실행 시 자동으로 생성됩니다.

## 권한

- 일반 사용자: 프로필 등록 및 조회
- 관리자: 모든 명령어 + 경고/메모 관리

## 문제 해결

### 봇이 실행되지 않을 때
1. Python 버전 확인: `python --version`
2. 필요한 패키지 설치 확인: `pip list`
3. `.env` 파일에 올바른 토큰이 입력되었는지 확인

### 명령어가 작동하지 않을 때
1. 봇이 서버에 제대로 초대되었는지 확인
2. 봇에 필요한 권한이 있는지 확인
3. 슬래시 커맨드 동기화 대기 (최대 1시간 소요 가능)
